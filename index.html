Що таке HTML?

HTML (HyperText Markup Language) - це стандартна мова розмітки для створення веб-сторінок.

HTML Редактори коду та IDE

Для написання HTML-коду можна використовувати різні редактори коду та IDE, такі як Visual Studio Code, Sublime Text, Atom, Brackets тощо.

HTML Основні приклади

Ось приклад простої HTML-сторінки:

        <!DOCTYPE html>
        <html>
        <head>
            <title>Title of the document</title>
        </head>
        <body>
            <h1>This is a heading</h1>
            <p>This is a paragraph.</p>
        </body>
        </html>

HTML Елементи

HTML елементи складаються з тегів, атрибутів та вмісту. Наприклад, <p>Текст параграфа</p>.

HTML Атрибути

Атрибути надають додаткову інформацію про елементи HTML. Наприклад, атрибут href в елементі <a> визначає URL-адресу посилання.
